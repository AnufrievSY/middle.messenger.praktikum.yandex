# Middle Messenger

Учебный проект мессенджера для Яндекс Практикума (Sprint 3).

## Реализованная функциональность

- SPA-роутер на History API с маршрутами:
  - `/` — вход,
  - `/sign-up` — регистрация,
  - `/settings` — профиль,
  - `/messenger` — чаты.
- Поддержка переходов кнопками интерфейса, а также через браузерные кнопки «назад/вперёд».
- Базовая защита роутов: неавторизованные пользователи перенаправляются на страницу входа.
- Полная авторизация через API:
  - регистрация,
  - вход,
  - выход,
  - получение текущего пользователя.
- Работа с пользователем через API:
  - обновление профиля,
  - смена пароля,
  - обновление аватара.
- Работа с чатами через API:
  - получение списка чатов,
  - создание чата,
  - добавление/удаление пользователя в чате.
- Подключение WebSocket для сообщений в реальном времени.
- Сборка проекта на Vite.

## Технологии и инструменты

- TypeScript
- Vite
- SCSS
- ESLint + Stylelint
- XMLHttpRequest (с `withCredentials`) для HTTP API
- WebSocket API
- Компонентный подход: `Block` + `EventBus`

## Установка и запуск

Установить зависимости:

```bash
npm install
```

Запуск проекта в режиме разработки:

```bash
npm run dev
```

Сборка проекта:

```bash
npm run build
```

Проверка типов и линтинга:

```bash
npm run lint
npm run lint:eslint
npm run lint:styles
npm run typecheck
```
